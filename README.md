# Best of Breed Technology Ltd - Corporate Portal

This repository contains the official corporate website for **Best of Breed Technology Ltd**, a premier IT solutions provider. The project focuses on a high-performance, responsive user interface designed to showcase technical services and corporate identity.

## 🌐 Live Demo
[View Live Site](https://bestofbreedtechnologyltd.co.ke)

## 🛠 Project Highlights

* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewing.
* **Modern UI/UX:** Implemented custom CSS scroll effects and a sleek dark mode toggle.
* **Service Showcase:** Dedicated sections for IT Support, Networking, and Software Solutions.
* **Performance Optimized:** Minimized asset loading times for a professional user experience.

## 💻 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Flexbox/Grid layouts)
* **Interactivity:** JavaScript (Scroll-reveal animations, responsive navigation)
* **Optimization:** SEO-friendly meta tagging and semantic HTML.

## 🚀 Key Features Included

- **Dynamic Header:** A responsive navigation system that adapts to user scroll behavior.
- **Service Cards:** Interactive UI elements detailing the company’s technical offerings.
- **Contact Integration:** Built-in lead generation/contact form for client inquiries.

## 📂 Repository Structure

- `/assets`: Contains optimized images and branding assets.
- `/css`: Custom stylesheets including the dark mode implementation.
- `/js`: Logic for navigation and UI animations.
